# Project Hedge · Field Guide

Mobile-first field reference for an off-grid homestead build near Ashland, OR.
Static site (no build step), installable as a PWA, works offline once loaded.

## Pages

| Page | What it covers |
|---|---|
| `index.html` | Home |
| `plot-plan.html` | Site water + electrical plot plan (shared trench, line schedule) |
| `battery-wiring.html` | 4P battery-to-bus wiring sheet (run schedule, torque, connect sequence) |
| `deck-layout.html` | To-scale deck box layout, stud map, cable cut schedule |
| `shed-freezer.html` | Dedicated 2P shed freezer system |
| `chicken-waterer.html` | 24V heated waterer spec |
| `litime-email.html` | LiTime's written answer on 6P (the record behind the 4P decision) |

## Notes

- Main bank is **4P** (LiTime rates this cell for max 4 in parallel). The other 2 batteries run the shed freezer.
- `deck-layout.html` is generated by a script that draws the SVG to scale **and** computes the cut
  schedule from the same coordinates, so the drawing and the numbers cannot disagree.
- Bump `CACHE` in `sw.js` whenever content changes, so installed phones fetch fresh pages.
- Served via GitHub Pages. All content is schematic. Verify measurements on site.
