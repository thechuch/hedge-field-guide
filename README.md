# Project Hedge — Field Guide

Mobile-first field reference for an off-grid homestead build near Ashland, OR.
Static site (no build step), installable as a PWA, works offline once loaded.

**Pages**
- `index.html` — home
- `plot-plan.html` — site water + electrical plot plan
- `battery-wiring.html` — 6× 24V battery-to-bus wiring sheet
- `chicken-waterer.html` — 24V heated waterer spec
- `litime-email.html` — draft email to LiTime (6P question)

Served via GitHub Pages. All content is schematic — verify measurements on site.
